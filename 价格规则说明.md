# 价格规则说明

本文档说明项目中价格相关的两张规则表分别负责什么，以及最终网页价格是如何计算出来的。

## 1. 两张规则表的职责

### `discount_rules.xlsx`

作用：

- 按商品标题关键词匹配折扣规则
- 用来计算“韩币折扣价”

主要字段：

- `keyword`：商品标题关键词，例如 `2526 BURTON`
- `discount_type`：折扣类型，当前支持 `rate` / `amount`
- `discount_value`：
  - `rate` 时：折扣率，例如 `0.7`
  - `amount` 时：直减金额，例如 `50000`
- `enabled`：是否启用，`1` 启用，`0` 停用
- `priority`：优先级，命中多条时取更高优先级
- `note`：备注

结果：

- 生成 `latest_discount_price`

---

### `pricing_rules.xlsx`

作用：

- 按商品分类设置人民币运费
- 设置韩元转人民币汇率
- 设置利润率

主要字段：

- `category`：商品分类，例如 `滑雪板`
- `exchange_rate`：韩元转人民币汇率，例如 `0.0052`
- `shipping_fee_cny`：人民币运费，例如 `400`
- `profit_rate`：利润率，例如 `0.10`
- `enabled`：是否启用，`1` 启用，`0` 停用
- `note`：备注

结果：

- 生成人民币价格相关字段

## 2. 最终价格计算顺序

项目中的价格计算顺序固定如下：

### 第一步：计算韩币折扣价

优先使用 `discount_rules.xlsx`

- 如果商品标题命中了折扣规则：
  - 计算出 `latest_discount_price`
- 如果没有命中：
  - `latest_discount_price` 留空

---

### 第二步：确定韩币基础价

用于后续换算人民币的韩币价格按下面顺序取值：

1. 优先取 `latest_discount_price`
2. 如果没有折扣价，则取 `price`

---

### 第三步：换算人民币基础价

公式：

```text
price_cny = 韩币基础价 * exchange_rate
```

---

### 第四步：计算网页展示现价

公式：

```text
final_price_cny = (price_cny + shipping_fee_cny) * (1 + profit_rate)
```

这个价格就是网页上展示的大字现价。

---

### 第五步：计算网页展示原价

网页划线原价使用数据库里的韩币原价 `original_price` 计算。

公式：

```text
original_price_cny = (original_price * exchange_rate + shipping_fee_cny) * (1 + profit_rate)
```

这个价格就是网页上展示的划线原价。

## 3. 数据库里的相关字段

价格相关字段主要在 `product_updates` 表中：

- `price`：网站抓到的韩币售价
- `original_price`：网站抓到的韩币原价
- `latest_discount_price`：按 `discount_rules.xlsx` 算出的韩币折扣价
- `price_cny`：韩币基础价换算后的人民币价格
- `shipping_fee_cny`：人民币运费
- `final_price_cny`：网页展示现价
- `original_price_cny`：网页展示原价
- `exchange_rate`：实际使用的汇率
- `profit_rate`：实际使用的利润率

## 4. 网页展示逻辑

网页当前价格展示规则如下：

- 大字现价：`final_price_cny`
- 划线原价：`original_price_cny`
- 运费信息：`shipping_fee_cny`

## 5. 以后维护时怎么改

### 如果你想改折扣

修改：

- `discount_rules.xlsx`

然后执行：

```bash
python3 refresh_cny_pricing.py
python3 export_all_stock.py
python3 generate_catalog_site.py
```

说明：

- 如果你修改的是折扣关键词或折扣比例，先重新跑价格刷新即可

---

### 如果你想改运费 / 汇率 / 利润

修改：

- `pricing_rules.xlsx`

然后执行：

```bash
python3 refresh_cny_pricing.py
python3 export_all_stock.py
python3 generate_catalog_site.py
```

---

### 如果你重新抓了网站数据

执行：

```bash
python3 update_stock_db_concurrent.py
python3 refresh_cny_pricing.py
python3 export_all_stock.py
python3 generate_catalog_site.py
```

## 6. 简单理解

一句话总结：

- `discount_rules.xlsx` 负责“折扣”
- `pricing_rules.xlsx` 负责“运费、汇率、利润”
- 网页最终显示的是“折后换算人民币后，再加运费和利润”的价格
