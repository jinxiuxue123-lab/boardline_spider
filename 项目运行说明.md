# 项目运行说明

本文档用于日常维护当前 `boardline_spider` 项目，说明每天需要执行的命令、每条命令的作用，以及在什么情况下需要补跑。

## 1. 当前项目整体作用

本项目用于抓取韩国滑雪装备网站 `boardline` 的商品信息，并完成：

- 分类商品列表抓取
- 商品图片保存
- 商品详情页价格与库存抓取
- 折扣价计算
- 人民币价格计算
- 全量库存导出
- 网页商品展示数据生成

当前数据库文件：

- `products.db`

当前网页目录：

- `web/`

## 2. 最常用命令

### 2.1 每天日常更新

每天最常用的是这一条：

```bash
python3 daily_sync.py
```

作用：

- 快速巡检新增商品
- 同步网站商品列表
- 更新商品上下架状态
- 更新商品价格和库存
- 导出当天变化商品
- 导出当天新增商品
- 打印当天汇总

适用场景：

- 正常日更
- 想检查今天有没有新增、价格变化、库存变化

---

### 2.2 刷新折扣与人民币价格

如果你修改了：

- `discount_rules.xlsx`

就执行：

```bash
python3 refresh_discount_pricing.py
```

作用：

- 不重新抓网站
- 重新按当前折扣规则计算韩币折扣价
- 同时联动重算所有商品的人民币价格

适用场景：

- 改了折扣规则
- 改了标题关键词
- 改了折扣比例

---

### 2.3 刷新人民币价格

如果你修改了：

- `pricing_rules.xlsx`

就执行：

```bash
python3 refresh_cny_pricing.py
```

作用：

- 不重新抓网站
- 只按当前分类运费、汇率、利润规则重算人民币价格

适用场景：

- 改了分类运费
- 改了汇率
- 改了利润率

---

### 2.4 导出全量库存表

```bash
python3 export_all_stock.py
```

作用：

- 导出当前数据库里全部商品的库存表
- 生成文件：
  - `all_products_stock.xlsx`

适用场景：

- 每天更新后想看最新全量商品数据
- 想给别人发库存表

---

### 2.5 刷新网页展示数据

```bash
python3 generate_catalog_site.py
```

作用：

- 从数据库生成网页展示数据
- 更新：
  - `web/catalog-data.js`

适用场景：

- 抓完商品后刷新网页
- 改完价格规则后刷新网页

## 3. 建议的每日执行顺序

### 方案 A：只做正常日更

```bash
python3 daily_sync.py
python3 refresh_discount_pricing.py
python3 export_all_stock.py
python3 generate_catalog_site.py
```

也可以直接用一键脚本：

```bash
./run_daily.sh
```

适合：

- 每天日常维护
- 想同时更新数据库、导出表格、刷新网页

---

### 方案 B：只改了价格规则，不重新抓网站

如果只修改了折扣：

```bash
python3 refresh_discount_pricing.py
python3 export_all_stock.py
python3 generate_catalog_site.py
```

如果只修改了运费 / 汇率 / 利润：

```bash
python3 refresh_cny_pricing.py
python3 export_all_stock.py
python3 generate_catalog_site.py
```

---

### 方案 C：只想刷新网页

```bash
python3 generate_catalog_site.py
```

适合：

- 数据库已经是最新
- 只是想重新生成网页

## 4. 每个脚本的作用

### `daily_sync.py`

日常更新主入口。

执行内容：

- `check_new_products.py`
- `spider_list_db.py`
- 上下架状态处理
- `update_stock_db_concurrent.py`
- `export_changes.py`
- `export_new_products.py`

---

### `spider_list_db.py`

抓取网站商品列表。

作用：

- 获取分类里的全部商品
- 保存商品基础信息到 `products`
- 下载商品图片
- 更新 `last_seen`

---

### `update_stock_db_concurrent.py`

抓取商品详情页。

作用：

- 获取价格
- 获取原价
- 获取库存
- 计算折扣价
- 计算人民币价格
- 记录价格变化和库存变化

---

### `repair_missing_products.py`

补齐缺失商品信息。

作用：

- 只补抓价格缺失、库存缺失、图片缺失的商品

适用场景：

- 导出表里有空白价格
- 网页里有商品没有图片
- 某次抓取中途失败后想补齐

命令：

```bash
python3 repair_missing_products.py
```

---

### `refresh_discount_pricing.py`

刷新折扣与人民币价格。

作用：

- 不重新抓网站
- 重新读取 `discount_rules.xlsx`
- 重算 `latest_discount_price`
- 同时联动重算人民币价格

---

### `refresh_cny_pricing.py`

刷新人民币价格。

作用：

- 不重新抓网站
- 只根据当前分类运费、汇率、利润规则重算人民币价格字段

---

### `export_changes.py`

导出当天变化商品。

生成文件：

- `stock_changes_YYYY-MM-DD.xlsx`

---

### `export_new_products.py`

导出当天新增商品。

生成文件：

- `new_products_YYYY-MM-DD.xlsx`

---

### `export_all_stock.py`

导出所有商品。

生成文件：

- `all_products_stock.xlsx`

---

### `generate_catalog_site.py`

生成网页展示数据。

更新文件：

- `web/catalog-data.js`

## 5. 规则文件说明

### `discount_rules.xlsx`

作用：

- 按商品标题关键词匹配折扣规则

修改后需要执行：

```bash
python3 refresh_discount_pricing.py
python3 export_all_stock.py
python3 generate_catalog_site.py
```

---

### `pricing_rules.xlsx`

作用：

- 按分类设置：
  - 汇率
  - 人民币运费
  - 利润率

修改后需要执行：

```bash
python3 refresh_cny_pricing.py
python3 export_all_stock.py
python3 generate_catalog_site.py
```

## 6. 如果遇到问题怎么办

### 情况 1：导出表里价格或库存空白

执行：

```bash
python3 repair_missing_products.py
python3 export_all_stock.py
python3 generate_catalog_site.py
```

---

### 情况 2：列表抓取中某些页面失败

查看：

- `failed_list_pages.txt`

说明：

- 当前系统会自动重试 3 次
- 仍失败会记录到这个文件

---

### 情况 3：详情页抓取失败

查看：

- `failed_stock_urls.txt`

然后执行：

```bash
python3 repair_missing_products.py
```

---

### 情况 4：网页价格没更新

如果你改的是折扣规则，执行：

```bash
python3 refresh_discount_pricing.py
python3 generate_catalog_site.py
```

如果你改的是运费 / 汇率 / 利润，执行：

```bash
python3 refresh_cny_pricing.py
python3 generate_catalog_site.py
```

## 7. 你每天最推荐执行的命令

如果按你现在的使用方式，最推荐的日常命令就是：

```bash
python3 daily_sync.py
python3 refresh_discount_pricing.py
python3 export_all_stock.py
python3 generate_catalog_site.py
```

含义：

1. 更新网站数据
2. 刷新折扣和人民币价格
3. 导出库存表
4. 刷新网页

这四条基本就是你每天的标准流程。
